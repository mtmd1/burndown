# Burndown

Generate burndown charts from Microsoft Planner xlsx exports with sprint tracking and configurable styling.

All code generated by Claude Opus 4.5-20251101.

## Setup

```sh
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Usage

Put a Planner .xlsx export in this directory.

```sh
python burndown.py              # all tasks
python burndown.py --sprints    # only tasks within sprint ranges
python burndown.py --sprint     # current sprint only
python burndown.py --compress   # compress inactive periods (threshold in config.toml)
```

Outputs `burndown.png` and `burndown.html`.

## Config

- `config.toml` - colours, line styles, sizes and time-skip threshold
- `sprints.csv` - sprint date ranges (see `example-sprints.csv`)
